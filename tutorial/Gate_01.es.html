Convirtiendo HTML en lluvia

En lluvia existe la clase Gate que mapea un elemento HTML en un elemento lluvia. Los elementos HTML se enlazan con lluvia a través de su id o, si no tiene, de su DOM o, si tampoco lo tuviera, lluvia crea el elemento de enlace.

Con esta clase podremos añadir efectos gráficos al elemento HTML que queramos.

INPUT TERMINAL

Your code: 
Button.prototype = new Gate
Button.prototype.constructor = Button

function Button(element){
	
	try {
		if (arguments.length)
			Gate.call(this, element)	// Call to the super constructor (it does all the work).
	} catch (e) {
		if ($K_debug_level >= $KC_dl.DEVELOPER)
			alert("No event handlers were found.\nException: " + e.toSource())
	}
}

Button.prototype.do_onclick   = function(event, element){
	alert("You have made click.")
}
a = new Button("debug")
run
