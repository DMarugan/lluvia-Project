Básico de hilos

Cualquier objeto derivado de Thread se registra automáticamente en el procesador y asume un método run.

El método run por defecto es abstracto y hay que redifinirlo (javascript nos avisará).

INPUT TERMINAL

Your code: 
			processor = new Processor()
			my_Thread = new Thread()
			processor.run()
run

OUTPUT CONSOLE

#debug

Conclusión

En ejemplos anteriores redefiníamos un método run para el hilo creado.

En esta parte, al pulsar sobre el botón run, javascript nos avisa del error ya que no hay ningún método redifinido para run de my_Thread.
