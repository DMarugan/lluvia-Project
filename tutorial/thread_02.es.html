Redefiniendo el método run

En el siguiente ejemplo se puede ver cómo redifinimos el método run de my_Thread.

INPUT TERMINAL

Your code: 
			processor = new Processor()
			function my_solicitor(processor_time){
			  var clock = document.getElementById('debug')
			  clock.innerHTML = processor_time.toString()
			}
			sol = my_solicitor

			my_thread = new Thread(my_solicitor, processor)
			processor.run()
run 
                          
                          if(processor)
                             processor.kill(my_thread, sol)
stop

OUTPUT CONSOLE

#debug

Conclusión

Al crear el objeto my_Thread pasamos como parámetro la dirección de memoria del método aquí redefinido 'function my_solicitor()'.

Cuando llamemos al método run de processor, esa dirección de memoria servirá para llamar a la función redefinida, en otras palabras, para llamar a la nueva función run de my_Thread.
