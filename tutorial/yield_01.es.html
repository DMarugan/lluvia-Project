Método Yield

Yield es un método que contienen todas las funciones en lluvia. Lo que hace Yield es ejecutar el último argumento recibido en una función sólo en caso de que este sea, a su vez, otra función.

INPUT TERMINAL

Your code: 
function f_example(a, b, c){
     
     return a + b + c + f_example.yield(3)
} 

alert( 
 f_example(1, 2, 3, function(e){ return e + 2 })
)
run

OUTPUT CONSOLE

#debug

Conclusión

En el ejemplo declaramos la función f_example. Cuando llamamos a la función creada pasamos los valores 1,2,3 y, como último parámetro, la dirección de memoria de una nueva función declarada ahí mismo. Cuando se ejecuta f_example se devuelve el valor de la suma de los tres primeros parámetros más el resultado que nos devuelva la ejecución del último parámetro de f_example que es una función (el método yield de f_example es quien ejecuta dicha función). Podemos ver cuando invocamos esta función, con yield, que enviamos como parámetro el valor 3 y que en la declaración de esta función se le suma 2 al parámetro recibido. Finalmente alert nos muestra por pantalla el resultado de todo este proceso.
